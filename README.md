# 🔗 URL Shortener Web App

A simple and responsive URL Shortening tool built using HTML, CSS, and JavaScript.  
It uses the TinyURL API to convert long URLs into short, shareable links.

---

## 🚀 Features

- 🔗 Shortens long URLs instantly
- 🌐 Uses real API (TinyURL)
- 📋 Copyable short links
- 💡 Simple and clean UI
- ⚡ Fast and lightweight (no backend)

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- TinyURL API

---

## 📸 Demo

1. Enter a long URL  
2. Click "Shorten"  
3. Get a shortened link instantly
